# Projet Data Mining 2026 – E-commerce

## Objectif
Ce projet vise à analyser les données clients d’un site e-commerce à l’aide
des techniques de Data Mining afin d’aider à la prise de décision marketing.

## Modèles utilisés
- Règles d’association (Apriori et FP-Growth)
- Segmentation par K-means
- Segmentation RFM

## Structure du projet
- notebooks/ : notebooks de modélisation
- app/ : application Streamlit
- reports/ : rapports du projet
- data/ : données brutes et préparées

## Application
L’application Streamlit permet :
- Le chargement des données
- Le choix du modèle d’analyse
- La visualisation interactive des résultats
